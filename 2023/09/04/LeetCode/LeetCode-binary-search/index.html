红蓝染色法

再来细细体会一下红蓝染色法，以例题为例：

34. 在排序数组中查找元素的第一个和最后一个位置

  • 给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。

  • 如果数组中不存在目标值 target，返回 [-1, -1]。

  • 你必须设计并实现时间复杂度为 $O(logn)$ 的算法解决此问题。

闭区间写法

对于两个指针 L 和 R

  • L 左侧 为 红色区域
  • R 右侧 为 蓝色区域
  • [L , R] 之间为 无色区域
  • 红色区域 和 蓝色区域 的判断条件是 互补的
  • 无色区域 表示 不确定区域
  • L <= R 均合法 不确定区间不为空

每次判断的索引为 $\lfloor \frac{L+R}2 \rfloor$

Shit Idea
对于这题，查找两次，一次查出第一个，另一次查出最后一个

对于第一个，判断条件为：

  • 红色: < target
  • 蓝色: >= target
  • L 左侧 均为 小于
  • R 右侧 均为 大于等于
  • 最终 L = R + 1，L 在 R 右侧
  • 那么 L 所指的有可能是答案

对于最后一个，判断条件为：

  • 红色: <= target
  • 蓝色: > target
  • L 左侧 均为 小于等于
  • R 右侧 均为 大于
  • 最终 L = R + 1，R 在 L 左侧
  • 那么 R 所指的有可能是答案

Shit Code

Good Idea
其实，与其查两次，每次条件不同，不如转换为每次查的数字不同

  • 第一次查 target
  • 第二次查 target + 1
    Ling’s Idea
